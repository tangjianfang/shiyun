# 诗云 · 端到端手动测试清单

> **目的**：在每次发布前，由人工按本清单逐项验证核心功能可用。
> **测试环境**：构建后的 `dist/诗云-学习版.html`（双击打开），Chrome 90+ / Edge 90+ / Firefox 88+。
> **测试耗时**：约 30-45 分钟（首次完整跑）。
> **回归频率**：每个版本发布前必跑；关键修复后跑相关章节。

---

## 0. 测试准备

- [ ] **0.1** 双击 `dist/诗云-学习版.html` 能正常打开（不报白屏）

## 4. 考核 - 4 种模式

> **前置**：当前用户至少学过 5 首诗

### 4A. 填空模式

- [ ] **4A.1** 进入「考核」tab → 模式选择「诗句填空」

- [ ] **4A.3** 下方字库显示 4-6 个字（含正确答案 + 干扰项）
- [ ] **4A.4** 点击字库中的字 → 填入挖空处
- [ ] **4A.5** 移动端触摸友好（字库大、点击响应快）
- [ ] **4A.6** 点「提交」→ 判分（正确率 0-100%）
- [ ] **4A.7** 错题显示正确答案 + 解释

### 4B. 选择题模式

- [ ] **4B.1** 模式选择「选择题」

- [ ] **4B.3** 4 个选项（1 正确 + 3 干扰）
- [ ] **4B.4** 点击选项 → 立即判分（绿色=对，红色=错）
- [ ] **4B.5** 显示正确答案 + 解析

### 4D. 听诗选诗模式
